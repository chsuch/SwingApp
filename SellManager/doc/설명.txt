<메뉴 설명>
1. 검색
	-구입내역 검색 : 
		a.검색조건(구입건수,고객명,전화번호) 선택 
			1)구입건수 : 입력된 숫자 이상 구입한 고객 검색(예:3을 입력하면 3건 이상 구입한 고객이 출력된다.)
			2)고객명,전화번호 : 고객명 또는 전화번호를 검색한다. 입력한 텍스트를 포함한 대상을 검색한다.
	-고객 검색
		a. 구입내역 검색과 동일

2. 입력
	-구입내역 입력 :
		a.저장 - 입력된 데이터를 DB에 저장한다.
		b.한줄추가 - 데이터를 추가 할 수 있도록 행을 추가한다.
		c.Excel - 제공된 입력form 에 맞추어 작성된 엑셀데이터를 불러온다.
		d.초기화 - 입력된 데이터를 초기화 한다.
		e.삭제 - 선택된 행을 삭제한다.(ctrl-key는 연속되지 않은 복수의 행을 선택, shift-key는 연속된 행을 선택)
	*주문인과 전화번호는 필수 입력사항

3. 편집
	-고객정보 수정
		a.검색 - 검색메뉴와 동일하게 동작
		b.저장 - 수정된 내용을 DB에 저장
		*주문인과 전화번호는 필수 입력사항
		*보라색으로 표시된 부분만 수정 가능
	-구매내역 수정
		a.검색 - 검색메뉴와 동일하게 동작
		b.저장 - 수정된 내용을 DB에 저장
		*보라색으로 표시된 부분만 수정 가능
	-DB초기화
		a.기 입력된 DB데이터를 초기화 한다.
	-내보내기
		a.기 입력된 데이터를 엑셀파일로 추출한다.
		*추축된 데이터는 [판매관리]프로그램이 있는 동일위치에 [backup]폴더 아래에 [판매이력_날짜.xls]이름으로 생성된다.
